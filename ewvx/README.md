# EWVX Library

Shared library for reading and writing EWVX (Ente Wurzel Video XML) files.

- **`ewvx::types`** — `EwvxData`, `EwvxMeta`, `EwvxFrame`
- **`ewvx::writer`** — Streaming v2.0 EWVX writer
- **`ewvx::parser`** — v2.0 EWVX parser with version validation

Part of [ewvx-converter](https://github.com/DemonicNomad/ewvx-converter).
